## License Summary

### MIT License
The majority of dependencies use the **MIT License**, which permits:
- Commercial use
- Modification
- Distribution
- Private use

With the requirement to include the license and copyright notice.

### Apache 2.0 License
Used by packages like Google API libraries and some UI components:
- google-genai
- google-generativeai
- google-auth
- recharts
- class-variance-authority

Permits commercial use, modification, and distribution with patent protection.

### BSD Licenses (2-Clause and 3-Clause)
Used by Django ecosystem and other libraries:
- Django
- djangorestframework
- asgiref
- etc.

Similar to MIT but with additional non-endorsement clause (3-Clause version).

### Other Licenses
- **ISC License**: lucide-react
- **MPL 2.0**: certifi
- **PSF**: typing-extensions
